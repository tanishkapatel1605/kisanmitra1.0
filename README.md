# 🌾 KisanMitra (किसान मित्र)
### AI Crop & Soil Advisory Chatbot for Indian Farmers

# Run The Deployed App Here: https://kisanmitraap.streamlit.app/

KisanMitra is an intelligent, high-performance agricultural assistant designed to provide practical, hyper-localized, and actionable farming advisory services to Indian farmers. Built with a futuristic **Cyber-Organic Midnight Dark** aesthetic, the application provides an optimized experience across both desktop and mobile devices.

---

## ✨ Features

* **Smart Agriculture Intelligence:** Powered by Groq's high-speed `llama-3.3-70b-versatile` model to deliver robust responses within seconds.
* **Regional Context Filters:** Allows farmers to input their specific Soil Profile (Sandy, Clay, Black Regur, etc.), Active Season (Kharif, Rabi, Zaid), and State to get customized, context-aware advice.
* **Bilingual Support:** Understands and responds seamlessly in both English and Hindi (including Hinglish).
* **Policy & Scheme Integration:** Automatically highlights relevant government initiatives like PM-Kisan, Fasal Bima Yojana, and Kisan Credit Card (KCC) when answering related problems.
* **Futuristic Responsive UI:** A premium neon midnight cyber-blue theme optimized for effortless thumb-navigation on mobile web browsers.
* **Instant Submission Mechanics:** Features a seamless, custom script to trigger message processing instantly on a single `Enter` keypress.
